Sprint Manager (Projeto de Programação Web)

Este é um projeto para a disciplina de Programação Web, focado em criar um sistema de gerenciamento de tempo (time-tracking) e sprints para projetos, permitindo o controle de horas por usuário e o registro de ponto.

Tecnologias Utilizadas

Conforme solicitado nas diretrizes do projeto:

Backend: PHP 8 (Nativo, sem frameworks)

Frontend: Javascript (Nativo, sem frameworks)

Banco de Dados: MySQL

Estilização: Tailwind CSS (via CDN, conforme permitido)

Arquitetura: PHP Tradicional com chamadas AJAX (sem API REST)

Como Executar o Projeto

Clone o repositório:

git clone 


Ambiente: Coloque a pasta sprint_manager dentro do htdocs do seu MAMP.

Banco de Dados:

Inicie o MAMP (verifique se o MySQL está na porta 8889).

Acesse o phpMyAdmin.

Crie um novo banco de dados chamado sprint_manager.

Instalação:

Abra o navegador e acesse: http://localhost:8888/sprint_manager/public/install.php

Isso criará todas as tabelas e o usuário admin.

IMPORTANTE: Após a instalação, DELETE O ARQUIVO public/install.php por segurança.

Acesse:

Acesse a tela de login: http://localhost:8888/sprint_manager/public/login.php
